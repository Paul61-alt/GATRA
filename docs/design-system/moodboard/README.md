## How to use this folder

When in doubt about a design decision:

1. Open this folder.
2. Find a similar pattern in a reference.
3. Use it as a starting point, not a copy.
4. If a reference doesn't exist for what you're designing, you might be inventing something — pause and ask "do we actually need this?"

---

## Adding a new reference

```
moodboard/
├── palantir-foundry-dossier.png
├── perplexity-research-mode.png
├── bloomberg-ticker-grid.png
└── ...
```

Filename pattern: `<source>-<screen-name>.<ext>`. All lowercase, kebab-case.

When adding, also note in a comment in this README what specifically you found useful in the reference (composition? color? typography?). This prevents the moodboard from becoming a graveyard of unexplained screenshots.
